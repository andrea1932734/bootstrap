My First Bootstrap Page

Column 1

a cosa serve bootstrap?

Bootstrap è un framework front-end open source che semplifica lo sviluppo web fornendo una griglia responsiva, componenti UI predefiniti, stili CSS di base e plugin Javascript. Facilita la creazione di interfacce utente coerenti e professionali, riducendo la necessità di scrivere manualmente ampie porzioni di codice.

Column 2

cosa devi importare per creare una pagina con bootstrap?

Per una pagina con Bootstrap HTML Base: File HTML con head e body. Bootstrap CSS: Importa il CSS di Bootstrap nell'elemento head. JavaScript Bootstrap: Aggiungi gli script JavaScript alla fine di /body.

Column 3

Cos’è un CDN? Che vantaggi comporta rispetto a scaricare bootstrap sul tuo sito?

Un CDN (Content Delivery Network) è una rete di server globale che fornisce file come CSS, JavaScript e immagini ai siti web. Utilizzando un CDN per Bootstrap anziché ospitare i file direttamente sul tuo sito, ottieni Velocità, Cache globale, Aggiornamenti automatici e Semplicità.

Column 4

A cosa serve questo attributo del tag meta?

L'attributo meta nel tag meta fornisce informazioni aggiuntive sulla pagina web, come descrizioni, parole chiave, encoding dei caratteri e istruzioni per i motori di ricerca. Aiuta i browser e i motori di ricerca a interpretare e presentare correttamente il contenuto della pagina.

Column 5

Che differenza c’è tra Container e Container Fluid (prova il container fluid creando una pagina chiamata BTS_1.1.1.html)

Container: Elemento di layout con larghezza fissa e margini laterali fissi, adatto per organizzare il contenuto in una griglia predefinita. Container Fluid: Elemento di layout che si estende per occupare l'intera larghezza del dispositivo, adattandosi a diverse dimensioni dello schermo senza margini laterali fissi, ideale per layout a larghezza intera.
